# 루틴덱

## 서비스 개요

게임 루틴을 만들고, 공유하고, 구독하면 사용자가 쓰는 LLM이 MCP를 통해 개인 게임 비서처럼 루틴을 조회·추천·체크할 수 있게 해주는 서비스

루틴덱은 두 가지 인터페이스를 가진다.

- **웹 인터페이스**: 사용자가 루틴을 만들고, 공유하고, 구독하고, 체크한다.
- **MCP 인터페이스**: 사용자가 쓰는 LLM이 루틴덱의 루틴과 진행 상태를 조회하고, 개인 게임 비서처럼 사용자를 돕는다.

루틴덱은 자체 AI를 운영하지 않는다. AI는 사용자가 이미 쓰는 ChatGPT, Claude, Cursor 같은 LLM 클라이언트가 담당한다.

---

## 해결하려는 문제

- 게임에 복귀했지만 지금 무엇부터 해야 할지 모른다.
- 여러 게임을 병행하면서 해야 할 일을 놓친다.
- 커뮤니티 공략글은 많지만 실제로 따라 하기 불편하다.
- 고인물, 길드장, 방송인은 같은 설명을 반복해야 한다.
- 사용자가 쓰는 LLM은 똑똑하지만, 사용자의 게임 루틴과 진행 상태를 알지 못한다.

---

## 핵심 도메인

| 도메인 | 설명 |
|---|---|
| `Routine` | 특정 게임·상황의 체크 가능한 가이드 묶음 (게임명 문자열로 관리) |
| `RoutineTag` | 루틴의 성격을 나타내는 태그 (복귀, 뉴비, 이벤트 등) |
| `RoutineItem` | 루틴 안의 개별 할 일 |
| `Subscription` | 사용자가 루틴을 구독한 정보 |
| `RoutineItemCheck` | 사용자별 루틴 아이템 체크 기록 |

---

## MVP 핵심 흐름

1. 루틴 생성
2. 루틴 아이템 추가
3. 공유 링크 생성
4. 공유 링크로 루틴 조회
5. 루틴 구독
6. 내 루틴덱에서 오늘/이번 주 할 일 확인
7. 루틴 아이템 완료/미완료 토글
8. MCP를 통한 루틴 조회 및 체크

---

## MVP 범위

### 웹

- 회원가입 / 로그인
- 게임명 입력 및 자동완성
- 루틴 생성 / 수정 / 삭제
- 루틴 태그 설정
- 루틴 아이템 생성 / 수정 / 삭제
- 루틴 공개 / 비공개 설정
- 공유 링크 조회
- 루틴 구독
- 내 루틴덱 조회
- 오늘 할 일 / 이번 주 할 일 조회
- 루틴 아이템 체크 / 체크 해제

### MCP (초기 read 중심)

- 공개 루틴 검색
- 공개 루틴 상세 조회
- 내 루틴덱 오늘 할 일 조회
- 내 루틴덱 이번 주 할 일 조회
- 루틴 구독 (write)
- 루틴 아이템 체크 / 해제 (write)

### 제외

- 로그인 / 회원가입 소셜 연동
- 루틴 복사 (구독 우선, 이후 확장)
- 알림
- 자동 초기화 스케줄러
- 루틴 댓글 / 좋아요
- AI 기능 (루틴덱 자체 AI 없음)
