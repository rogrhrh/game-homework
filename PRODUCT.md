# 겜숙제

## 서비스 개요

게임 유저가 일일/주간/월간 숙제표를 만들고, 공유 링크로 다른 사람에게 쉽게 공유할 수 있는 서비스다.
다른 사용자는 공유된 숙제표를 복사해서 자기 체크리스트로 사용할 수 있다.

---

## 해결하려는 문제

게임마다 일일/주간 숙제, 이벤트, 보스, 재화 수급 할 일이 흩어져 있고,
유저들은 구글시트, 메모장, 커뮤니티 글로 직접 숙제표를 만들어 쓰는 경우가 많다.
하지만 공유, 복사, 체크 상태 관리가 불편하다.

---

## MVP 핵심 흐름

1. 숙제표 템플릿 만들기
2. 숙제 항목 추가하기
3. 공유 링크 생성하기
4. 공유 링크로 숙제표 보기
5. 공유된 숙제표를 내 체크리스트로 복사하기
6. 내 체크리스트에서 항목 완료/미완료 체크하기

---

## MVP 범위

### 포함

- 숙제표 템플릿 생성
- 항목 반복 주기 설정 (DAILY / WEEKLY / MONTHLY / ONCE)
- 공유용 slug 생성
- 공유 템플릿 조회
- 템플릿 복사
- 내 체크리스트 조회
- 체크리스트 항목 토글

### 제외

- 로그인 / 회원가입
- 결제
- 알림
- 자동 초기화 스케줄러
- 게임별 공식 데이터 연동
- AI 기능
- 복잡한 권한 관리
